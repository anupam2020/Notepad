# Notepad
Organize your thoughts and ideas with this fast, free and lightweight notes app.

### Libraries :
* Lottie Animations
* Firebase
* StatusBarUtil
* Picasso
* Circle ImageView
* Dynamic Toast

### App Link :
https://play.google.com/store/apps/details?id=com.sbdev.notepad_project
